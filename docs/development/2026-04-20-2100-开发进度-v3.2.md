## 2. 已完成功能清单

### 2.1 核心能力
- [x] 统一 `ArkClient`（单例 + 懒加载 per-provider OpenAI client）
- [x] 多 Provider 抽象：`ark` / `dashscope`（自动按 `provider` 路由 base_url + api_key）
- [x] ModelConfig 注册表机制：`supports_thinking` / `output_format` / `requires_system_prompt` / `vl_high_resolution`
- [x] 官方推荐 prompt 格式按模型动态切换（方案 A，`@src/coord_formats.py`）
- [x] thinking matrix 自动组合（`--thinking-matrix`）
- [x] 并行执行（`ThreadPoolExecutor`，默认并行 4）
- [x] 单任务超时（`--task-timeout`，默认 120s）
- [x] Ground Truth 坐标评估（point-in-bbox ±5 容差 + IoU ≥ 0.3 兜底）
- [x] 多目标评估：Precision（点命中率）+ Recall（GT 覆盖率）双指标
- [x] 坐标预测解析器：已兼容 **9 种**输出变体（A/B/C/D/D'/E/F/G/H）
- [x] 可视化：标记预测点到原图 + 绘制 GT bbox
- [x] Markdown 报告 + JSON 原始结果自动生成
